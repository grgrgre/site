# NotebookLM Sources (Safe)

## Важливо
- `grgrgre/site` зараз `private`. Для імпорту через URL у NotebookLM потрібен `public` репозиторій, або завантаження файлів вручну.
- Нижче лише безпечні посилання. Секрети (`.env`, `.env.ftps`, `storage/data/admin-auth.json`, `storage/data/svityazhome.db`) навмисно не включені.

